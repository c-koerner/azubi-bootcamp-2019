## Installationsvoraussetzungen

* `brew` (MacOS)
* `awscli`
* `terraform`
* `node`

Zur Installation im Terminal ausführen

```bash
git clone https://github.com/spring-media/azubi-bootcamp-2019.git
cd azubi-bootcamp-2019/setup
./yolo.sh
```
